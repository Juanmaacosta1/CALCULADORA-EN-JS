# 🧮 Calculadora Web en JavaScript

Aplicación web interactiva que permite realizar operaciones matemáticas básicas directamente desde el navegador.
Desarrollada utilizando JavaScript puro para practicar lógica de programación y manipulación del DOM.

---

## 🌐 Probar la calculadora

👉 https://juanmaacosta1.github.io/CALCULADORA-EN-JS/

(No requiere instalación)


---

## 🚀 Funcionalidades

* Suma
* Resta
* Multiplicación
* División
* Limpieza de pantalla
* Entrada mediante botones interactivos

---

## 🧠 Lógica implementada

* Manejo de eventos de botones
* Construcción dinámica de operaciones
* Evaluación de expresiones matemáticas
* Validación básica de entrada

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript puro (Vanilla JS)

---

##  Ejecutar localmente

1. Clonar o descargar el repositorio
2. Abrir `index.html` en el navegador

No necesita servidor ni dependencias.
